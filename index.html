How it works

  1. We run a lightweight heuristic that checks texture/edges/noise and metadata.
  2. The result is a score from 0–100%. Scores ≥ 90% are flagged as AI generated.
  3. This runs entirely in your browser — no files are uploaded.

  • Downloadable report (TXT)
  • Image preview and basic metadata

Notes

This is a lightweight heuristic detector (B1). It's fast and easy to run locally but not as accurate as large trained models.
